## Supported platforms：

- Windows(386\64)
- macOS
- Linux（amd64、armv7\armv8）
- and More(You can compile it yourself).

### Install：
Download [the latest release for your system](https://github.com/yumenaka/comi/releases ) 

### Usage：
```
comi [flags] file_or_dir
#more：
comi --help
```
